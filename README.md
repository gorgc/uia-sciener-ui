# UIA Sciener Frontend

UIA Sciener Frontend using the [Vue + Typescript + Vite + Electron](https://github.com/vitejs/vite) framework.

## Installation

```bash
# Install dependencies
$ yarn

# Use example files
$ cp .env.example .env
```

## Running the app (Web)

```bash
# Development
$ yarn dev

# Production
$ yarn build
```

## Running the app (Electron)

```bash
# Development
$ yarn electron:dev

# Production
$ yarn app:build
```
